# Fiber Route Map

A comprehensive system for mapping and managing fiber optic routes.

## Project Structure
- `website/`: Premium Next.js 16 Frontend (Glassmorphism, Unified Multi-Theme UI, Multi-Identifier Auth, RBAC Management, Responsive Dashboard, Enhanced User Management).
- `node_backend/`: Node.js based REST API (Express, TypeScript, Knex) with Granular RBAC, Interactive Swagger Docs, and Health Monitoring.
- `(Other components will be listed here as they are developed)`

## Prerequisites
- Node.js 18+
- MySQL / MariaDB

## Getting Started
1. Setup the backend: See [node_backend README](node_backend/README.md) (runs on port 3001).
2. Setup the frontend: See [website README](website/README.md) (runs on port 3000).
